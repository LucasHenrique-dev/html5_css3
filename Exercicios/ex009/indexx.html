Outras Formatações

Código-fonte/Pré-formatação

Para você que quer escrever um código, mas esta cansad@ da formatação básica e padrão do HTML, tente agora mesmo usar a tag semântica <code>. Compare a seguir a diferença:


→ Sem a tag: System.out.print("Hello World");

→ Com a tag: System.out.print("Hello World");


Unindo-se com a tag semântica <pre> você poderá personalizar seu próprio bloco de código, uau!

        
    public int factorial(int number){
        if (number == 1) return 1;
        return number * factorial(number - 1);
    }

Citação simples

Remete a fala de alguém ou uma obra por meio da tag semântica <q>.

Ex.: Como diria Ben Parker: Com grandes poderes vem grandes responsabilidades.

Citações completas

É uma citação maior, em forma de bloco, que pode conter a referência para a citação em questão. Usa-se a tag semântica <blockquote>.

Ex.: Trecho adaptado do poema For Katrina's Sun-Dial de Henry Van Dyke:

O tempo é muito lento para os que esperam Muito rápido para os que têm medo Muito longo para os que lamentam Muito curto para os que festejam Mas, para os que amam, o tempo é eterno.

Abreviações

Serve para mostrar o significado da abreviação quando o mause pousa sobre a palavra. Para isso, utiliza-se a tag semântica <abbr>.

Ex.: Passe por cima das siglas HTML e CSS e descubra seus significados.

Texto Invertido

Com a tag semântica <bdo> é possível determinar a ordem de escita do texto, ou seja, se ele será lido da esquerda para a direita ou da direita para a esquerda.

Exs.:
Esse teste é um exemplo de frase escrita da esquerda para a direita.

Já esse é do tipo normal/habitual, ou seja, da esquerda para a direita.
