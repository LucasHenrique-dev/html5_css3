Principais Formatações

Negrito/Destaque

Atualmente no HTML5 você pode usar o negrito através da tag <b>, porém esta já é uma tag obsoleta (não semântica).

Desse modo é preferível usar a tag <Strong> que é a tag semântica para o negrito/destaque.

Itálico/Ênfase

Para o Itálico também ocorre situação parecida, onde a tag <i> pode sar usada para proporcionar o tal efeito, mas não é considerado semântico e, portanto, uma tag obsoleta.

Com o objetivo de contornar tal problema há <em> que é a tag semântica para o Itálico/Ênfase.

Texto Marcado

Para deixar o texto marcado (usando a propriedade style ⇒ Característica para essa tag em específico), como se estivesse com marca texto é necessário usar a tag <mark> desse jeito (usando seletor, CSS, no próprio documento ⇒ Característica vai para todos os elementos mark do documento em questão).

Texto pequeno

Para deixar um texto em letras miúdas é necessário usar a tag semântica <small>. Já para letras grandes há a tag obsoleta <Big> que realiza algo desse tipo.

Texto Deletado

Um texto deletado é um texto segredo gamer riscado que pode ser visulizado, mas não deve ser considerado.

Para criar um é necessário o uso de uma tag semântica <del>.

Texto Inserido

Um texto considerado inserido possui um sublinhado e para isso usa-se uma tag semântica <ins>.

Texto Sobrescrito

É bastante útil usar um texto sobrescrito para descrever fórmulas ou expressões matemáticas. Isso usando a tag semântica <sup>.

Ex.: ∫ 8x7 + 3x2 + 1 dx = x8 + x3 + x + c

Texto Subscrito

Para os textos subscritos a sua utilização mais notória é para descrever fórmulas químicas, utilizando para isso a tag semântica <sub>.

Ex.: CH4 + 2O2 2H2O + CO2 (Combustão do Metano).
